# Sales Trend Analysis Using MySQL

## Objective
To analyze monthly sales revenue and order volume using SQL aggregation functions.

## Tools Used
- MySQL

## Key Concepts
- GROUP BY year and month
- SUM() for revenue calculation
- COUNT(DISTINCT order_id) for order volume
- ORDER BY and LIMIT for sorting and filtering

## Outcome
Identified monthly sales trends and high-performing periods.
